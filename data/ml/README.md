ML artifacts

This folder contains results and models produced by the ML scripts in src/ml.

- classifier_LR/: baseline classifier outputs (results, plots)
- classifier_RF/: advanced classifier outputs (models, results, plots)

Reproduce or extend via:
- Training/evaluation: src/ml/classifier_LR.py, src/ml/classifier_RF.py
- Inference with saved models: src/ml/predict_with_saved_models.py

Note: Files are published as-is for reproducibility; rerun scripts to regenerate.
